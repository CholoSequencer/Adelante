so \((-1-\lambda)^2 - \frac{mn}{4} = (1 - \frac{mn}{4}) + 2\lambda + \lambda^2 = 0 \) s.t. \(\lambda = -1 \pm \frac{\sqrt{mn}}{2}\)
Since \(\lambda = -1 - \frac{\sqrt{mn}}{2}\) is always negative, the system is asymptotically stable when 4 > mn (both e-values are negative) and unstable when mn > 4 (one negative and one positive). If mn = 4 (s.t. e-values are one negative and one positive), then it is degenerate though stable since 0 > trace = -2.
